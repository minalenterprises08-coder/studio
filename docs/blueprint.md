# **App Name**: Minal Enterprises Online

## Core Features:

- Product Showcase: Display Minal Enterprises' offerings: custom printed boxes, labels, cards, and packaging accessories.
- Interactive Customizer: Interactive tool for custom boxes. Allows users to define size, material, finishing, and printing options for an instant price estimate.
- Quote Request: Form submission for requesting custom quotes for bulk orders and specialized printing.
- Portfolio Gallery: Display customer case studies and before/after showcases, each of the showcases consists of materials, finishing, qty and lead time to inspire trust in the services of Minal Enterprises.
- Contact and Support: Provide contact form and Whatsapp interaction to address immediate custom requests.
- AI-Powered Dieline Check: Allowing the user to upload a dieline which then gets validated by a generative AI tool to check for the dimensions as well as some common design problems with dielines.
- SEO and Analytics: Employ technical Search Engine Optimization standards. User tracking is enhanced by adding event triggers.

## Style Guidelines:

- Primary color: Very dark navy (#0F172A) for a premium feel.
- Accent color: Warm amber (#D97706) for CTAs and highlights, reminiscent of foil accents.
- Background color: Light off-white (#FFFDF8) with subtle paper texture for a tactile feel.
- Headline font: 'Playfair', a serif font that conveys elegance.
- Body font: 'Inter', a clean sans-serif font for readability.
- Minimalist icons, with amber accents, to represent product categories and services.
- Responsive 12-column grid layout adapting to desktop, tablet, and mobile.
- Subtle animations including parallax scrolling, hover effects, and animated transitions to enhance user experience.
